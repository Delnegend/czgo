# czgo

## Build from source
```bash
git clone https://github.com/Delnegend/czgo.git
cd czgo
go build -o czgo
```

## Binary releases
Check [releases](https://github.com/Delnegend/czgo/releases) for binary releases.

## Install from source
```bash
go install github.com/Delnegend/czgo@latest
```
